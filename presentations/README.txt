HYTEK OPERATIONS PLATFORM — MANAGER PRESENTATION
==================================================

FILE: hytek-ops-platform-2026.pptx

16 slides, self-running, 4 minute loop.
Brand: HYTEK yellow + black, Impact headlines.


HOW TO PLAY IT (on any PC with PowerPoint)
------------------------------------------
1. Double-click the .pptx file to open in PowerPoint.
2. Press F5 (or "Slide Show" > "From Beginning").
3. It runs itself — 15 seconds per slide, loops forever.
4. Press Esc to stop.

The file is pre-configured as a kiosk loop — you don't need
to set anything up on the meeting room PC.


OPTIONAL POLISH (2 minutes if you want it)
------------------------------------------
Add a Morph/Fade transition between slides:
  1. Open the file in PowerPoint
  2. Click the first slide in the left panel
  3. Ctrl+A to select all 16 slides
  4. Transitions tab > pick "Morph" or "Fade"
  5. Save

Morph needs Microsoft 365 / PowerPoint 2019+. Fade works everywhere.


BACKUP: EXPORT TO MP4 VIDEO
---------------------------
If you want a video file that plays on anything:
  File > Export > Create a Video
  - Use 1080p
  - "Use Recorded Timings and Narrations" (it'll use the 15s timings)
  - Click "Create Video", save as hytek-ops-platform-2026.mp4

About 4 minutes of video. Plays in any media player.


TO REBUILD OR EDIT
------------------
Script:  build.js (pptxgenjs)
Run:     cd presentations && node build.js
(then re-run the kiosk config PowerShell if you regenerate)
